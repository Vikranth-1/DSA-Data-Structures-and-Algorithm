Step 1: If START == NULL then UNDERFLOW
Step 2: Set PTR = START, PREPTR = NULL
Step 3: Traverse PTR until PTR->DATA == NUM or PTR->NEXT == START
         Update PREPTR = PTR, PTR = PTR->NEXT
Step 4: If PTR->DATA != NUM then
          Value not found, exit
Step 5: If PTR == START then  (deleting first node)
          Call Delete First Node algorithm
          Exit
Step 6: Set PREPTR->NEXT = PTR->NEXT
Step 7: Free PTR
Step 8: End
